# Missile Command Overdrive

A single-file HTML5 reimagining of Missile Command with enhanced visuals, audio, enemy variety, and experimental superweapons.

## Play

- Live URL: `https://somaliskinnyai.github.io/missile-command-overdrive/`
- Local: open `misslecommand_enhanced.html` in a modern browser.

## Controls

- `LMB`: Fire interceptor
- `H`: Toggle Hell Raiser deploy/retract
- `T`: Toggle visual theme (Modern/Xbox/Recharged)
- `C`: Toggle auto defense
- `F8`: Toggle debug telemetry
- `F9`: Export current wave telemetry JSON
- `F10`: Export full session telemetry JSON
- `E` or `RMB`: EMP pulse
- `M`: Mute
- `+` / `-`: Volume up/down
- `]` / `[` (or `PageUp` / `PageDown`): Skip level
- `1-5`: Shop upgrades (during shop)
- `R`: Restart

## Repository Structure

- `misslecommand_enhanced.html`: main game implementation (single-file runtime)
- `index.html`: GitHub Pages entry launcher
